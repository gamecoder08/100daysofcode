Problem Statement: To find arrange an array according to colors together. The nums `0`, `1` and `2` describe the colors white, red and blue respectively.

For example:

```
Input: nums = [2,0,2,1,1,0]
Output: [0,0,1,1,2,2]
```

Language Used: Python

Algorithm used:

There is not much to do here. Just normal sorting is required which can directly be achieved by `sort()` function in python.

Code Performance:

Runtime: 33ms
Memory: 16.34MB

NOTE: These performance results are for one time run and may give different stats for different inputs and runs.
