Problem Statement: To transpose a matrix.

For example:

```
Input: matrix = [[1,2,3],[4,5,6],[7,8,9]]
Output: [[1,4,7],[2,5,8],[3,6,9]]
```

Language Used: Python

Algorithm used:

To approach the matrix vertically instead of horizontally. Then, we save the vertical matrix in a new matrix as the horizontal and iterate over each column same way.

Code Performance:

Runtime: 48ms
Memory: 17.44MB

NOTE: These performance results are for one time run and may give different stats for different inputs and runs.
