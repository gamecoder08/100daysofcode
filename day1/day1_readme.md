Problem Statement: To create a program which upon receiving two arrays, determines whether the word contained by the arrays is same or not.

For example:

```
word1=["Progra","mmer"]
word2=["Pro","grammer"]
Output=True
```

Language Used: Python

Algorithm used:

To test whether both arrays have same word, we run through both loops once. We concate each element of array to two different empty strings. Then, we compare the both strings and return the True or False depending on whether they matched or not.
